Intra Kampus

Untuk himpunan mahasiswa di dalam kampus, saya mengikuti himpunan mahasiswa jurusan saya, yaitu HIMA IF. Di HIMA IF saya termasuk ke dalam divisi HUMAS, dan menjabat sebagai sekretaris. Di divisi ini saya dapat menambah relasi atau berjumpa dengan beberapa mahasiswa di jurusan lain, ataupun mahasiswa di luar kampus. Tak hanya tergabung dalam HIMA Informatika, saya juga mengikuti beberapa UKM di kampus, yaitu UKM Basketbal Teknokrat dan UKM Progaming Teknokrat, dan di UKM Protek saya tergabung di divisi Web Developer.

Extra Kampus

Untuk himpunan mahasiswa di luar kampus, saya mengikuti himpunan mahasiswa ilmu komputer seluruh Indonesia, yaitu PERMIKOMNAS. Di PERMIKOMNAS saya menjabat sebagai Bendahara Wilayah tersebut. Tak hanya Permikomnas, pada kesempatan kemarin, saat saya mengikuti program pertukaran mahasiswa merdeka Batch 4, saya berkesempatan menjadi Wakil Kepala Suku dari Inbound Universitas Musamus Merauke.
